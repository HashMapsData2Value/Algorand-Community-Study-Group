# Exercises - ECC

## Preparation

Make sure you have python installed.

Create virtual environment called venv:
- `python3 -m venv venv`

Activate the virtual environment:
- `. venv/bin/activate`

Install the packages into your virtual environment:
- `pip3 install -r requirements.txt`
